# MARIN Core
ICHIRO Controller Manager Board for Dynamixel
## Features

* RS485/TTL on 4 Buses
* BNO055 IMU Sensor With Absolute Orientation
* Hot Swap switching System for Mini PC (Intel NUC 10th)
* Motor Voltage Power Switch System
